Workout Application-'Get That.'

Get That.

  • Splash
  • Quiz
  • Calendar
  • Workouts
  • Videos

My favorite project thus far; this is a workout web application created using Ruby on Rails. I along with four other students created this application aimed at workout beginners. This application features individual user accounts where each user starts by creating a profile and answering a 10 question quiz. The user's quiz data is then analyzed and a customized workout is created. A custom calendar for each user is then populated based on creation date. Our application features videos showing correct execution of each move as well as detailed descriptions and tips. There is a user review page as well where users can rate and describe their workouts, allowing for communication between users in the platform.

Note: this application is not live.

Smart Shopper Extension

  • Toolbar
  • Adding
  • Digest

Working against the clock at a hackathon hosted at Ohio State, I along with two friends created a Google Chrome extension we felt was useful for the everyday shopper. Using data collected from Facebook friends, we found that there is a large percent of online shoppers who avoid buying something because the price is too high. Our extension allows users to select products with the click of a button where the information about the product is then saved. Users can save as many products as their hearts' desire. Then every week a price comparison is performed and if lower than at the time of selection, is added to a digest. This digest is then sent to the user.
